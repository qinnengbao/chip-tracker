# 筹码追踪软件 📱

一款专业的股票筹码分析工具，追踪主力筹码、机构筹码、散户筹码分布。

## ✨ 功能特性

- 📊 **筹码分布图** - 可视化展示不同价格区间的筹码堆积
- 👥 **股东户数趋势** - 追踪筹码集中/分散变化
- 💰 **主力成本分析** - 估算主力建仓成本区间
- 📈 **支撑压力位识别** - 根据筹码峰判断关键价位
- ⭐ **自选股管理** - 添加关注股票，实时追踪

## 📥 下载安装

### 方式一：从Releases下载（推荐）

1. 点击 [Releases](https://github.com/qinnengbao/chip-tracker/releases)
2. 下载最新的 `app-release.apk`
3. 手机上点击安装

### 方式二：自动编译

每次代码更新都会自动编译APK，可在 Actions 页面下载。

## 🚀 使用说明

1. 打开App，搜索股票代码或名称
2. 添加到自选股
3. 查看筹码分布、股东户数变化
4. 分析主力动向，辅助投资决策
